# バリューチェーン・利益プール分析 PowerPoint ジェネレーター

業界全体のバリューチェーンを段階別に分解し、**利益プールの分布**と**対象会社のポジション**を
1枚のスライドで可視化するコンサル定番の戦略分析スキル。

「事業のどこで儲けているか」「対象会社の強み・弱みはチェーン上のどこか」を示すことで、
**戦略的な資源配分の意思決定を支援**する。

---

## スライド構成

| セクション | 位置 | 内容 |
|---|---|---|
| **メインメッセージ** | 最上部 | バリューチェーン分析からの示唆を1文で。最大70文字 |
| **チャートタイトル** | メインメッセージ直下 | 「バリューチェーン・利益プール分析：○○業界」 |
| **シェブロン（矢印）** | 中央上部 | 5〜7段階のバリューチェーン段階を横並び表示 |
| **詳細パネル** | 中央下部 | 各段階の利益プール率バー + 対象会社ポジション(強/中/弱) |
| **示唆ブレット** | 下部 | バリューチェーン分析からの戦略的示唆 |
| **出典** | 最下部 | 情報ソースの記載 |

### シェブロン部分の構成

- **段階名** (Bold 13pt 白): 例「原材料調達」「完成車組立」「アフターサービス」
- **活動内容** (10pt 白): 各段階の具体的活動の補足
- **背景色**: 紺の濃淡グラデーション（左→右で薄くなる）

### 詳細パネル（各段階）

- **利益プール率**: パーセント表示 + 横バー（最大値基準でスケーリング）
- **対象会社ポジション**:
  - **●●●  強み**（緑 #1B7A3B）: 業界内で優位
  - **●●○  中位**（オレンジ #DA7A2D）: 平均的
  - **●○○  弱み**（赤 #B83A3A）: 劣位

利益プール率は色分け（自動）:
- 25%以上: 濃緑（高利益プール）
- 10-25%: オレンジ
- 10%未満: 赤

---

## 入力パターンと処理フロー

### パターンA：業界情報・収益構造の生データから分析

1. **Step 1: 業界全体のバリューチェーンを5〜7段階に分解**
2. **Step 2: 各段階の利益プール率を推計**（業界レポート・有報の原価率分解から）
3. **Step 3: 対象会社の各段階での競争力を評価**（強/中/弱）
4. **Step 4: Markdownでユーザーに提示**し、確認・修正を求める
5. **Step 5: 承認後**、PowerPointを生成する

### パターンB：データが整理済みで入力された場合

1. **Step 1: 内容を確認**
2. **Step 2: 確認後、PowerPointを生成する**

### バリューチェーン分解の典型パターン

| 業界 | 典型的な段階 |
|---|---|
| 自動車 | 原材料 → 部品 → 完成車組立 → 流通 → 販売 → アフターサービス → 金融 |
| 食品・飲料 | 原料調達 → 加工製造 → 包装 → 流通 → 小売 → ブランディング |
| 医薬品 | R&D → 臨床試験 → 製造 → 流通 → 営業（MR） → 薬局・処方 |
| ソフトウェア | 開発 → 配布 → サポート → カスタマイズ → サブスクリプション |
| 建設 | 設計 → 資材調達 → 施工 → 検査 → アフターメンテナンス |

## JSONデータ仕様

`{{WORK_DIR}}/value_chain_data.json` に以下の形式で保存する：

```json
{
  "source": "出典：業界レポート（富士経済）、対象会社有報、各社IR資料（2024年度ベース）",
  "main_message": "業界の利益プールは川下に偏在しており、対象会社は川上偏重から川下へのシフトを加速すべき",
  "chart_title": "バリューチェーン・利益プール分析：自動車業界",
  "stages": [
    {
      "name": "原材料調達",
      "activity": "鉄鋼・樹脂・電子部品",
      "profit_pool_pct": 5,
      "position": "weak"
    },
    {
      "name": "部品製造",
      "activity": "エンジン・電池",
      "profit_pool_pct": 12,
      "position": "medium"
    },
    {
      "name": "販売・ディーラー",
      "activity": "新車販売・顧客対応",
      "profit_pool_pct": 25,
      "position": "strong"
    },
    {
      "name": "アフターサービス",
      "activity": "メンテナンス・修理",
      "profit_pool_pct": 30,
      "position": "weak"
    }
  ],
  "implications": [
    "利益プールの55%が川下（販売+アフターサービス）に集中",
    "対象会社は販売で強みを持つが、アフターサービスで弱み",
    "川上投資への資源シフトが正当化される"
  ]
}
```

### JSONフィールド仕様

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `source` | string | 任意 | 出典テキスト |
| `main_message` | string | 必須 | メインメッセージ（最大70文字） |
| `chart_title` | string | 任意 | チャートタイトル |
| `stages` | array | 必須 | バリューチェーン段階配列（5〜7段階推奨、最大7） |
| `stages[].name` | string | 必須 | 段階名（簡潔に。例：「販売・ディーラー」） |
| `stages[].activity` | string | 任意 | 活動内容の補足 |
| `stages[].profit_pool_pct` | number | 任意 | 利益プール率（%）。null可 |
| `stages[].position` | string | 任意 | 対象会社のポジション："strong"/"medium"/"weak" or "強"/"中"/"弱" |
| `implications` | array | 任意 | 示唆のブレット項目（2〜4項目推奨） |

### `position` の受付値

| 評価 | 受付値 |
|---|---|
| 強み | "strong", "high", "強" |
| 中位 | "medium", "mid", "中" |
| 弱み | "weak", "low", "弱" |

## 注意事項

- **段階名は短く**: シェブロン形状の制約で、段階名は10文字以内が推奨
- **利益プール率の根拠**: 業界レポート（富士経済・矢野経済）や有報の原価率内訳から推計
- **ポジション評価の客観性**: 競合との比較データに基づき、定性判断を最小限に
- **業界によって最適な段階数が異なる**: 5段階で十分な業界もあれば、7段階必要な業界も
- **「示唆」が肝**: 単なる事実列挙ではなく、戦略的アクションにつながる示唆を3個前後

## 企業調査レポートでの使用例

### 配置位置（推奨）

戦略的示唆セクションの一部として配置:

```
1. エグゼクティブサマリー
2-9. 対象会社・市場・競合の各種分析
10. バリューチェーン・利益プール分析 ← 本スキル
11. 成長ドライバー分析
12. 推奨アクション
```

### 価値の出し方

「マクロ分析」「市場分析」「競合分析」が**現状把握**であるのに対し、
バリューチェーン分析は **「戦略的どこで戦うか」を導く出口分析**。
事実分析の最後、推奨アクションの前に配置すると、論理展開が綺麗になる。
